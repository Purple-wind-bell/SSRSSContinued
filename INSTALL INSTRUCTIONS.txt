INSTALL INSTRUCTIONS for SSRSS

1. Download and install RSS:    https://github.com/KSP-RO/RealSolarSystem/releases/tag/v12.0.0

2. Download and install RSS TEXTURES:   https://github.com/KSP-RO/RSS-Textures/releases

3. Install the SSRSS folder into your GameData folder

OPTIONAL: These mods are currently supported and just need to be downloaded and installed.

-Environmental Visual Enhancements 

-Scatterer

-Distant Object Enhancements

-PlanetShine
